# AI Interview Report Generator

An AI-powered web application that analyzes a candidate's resume and job description to generate an interview preparation report and an optimized resume.

## Features

* User Registration & Login (JWT Authentication)
* Resume PDF Upload
* AI-Generated Interview Report
* Match Score Analysis
* Technical Interview Questions
* Behavioral Interview Questions
* Skill Gap Identification
* Preparation Plan Generation
* AI-Generated Resume PDF
* MongoDB Database Integration

## Tech Stack

### Frontend

* React.js
* Vite
* SCSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer

### AI

* Google Gemini API

## Installation

### Clone Repository

```bash
git clone https://github.com/mg6129327-boop/interview-website.git
cd interview-website
```

### Backend Setup

```bash
cd project/Backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd project/Frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside Backend folder:

```env
GOOGLE_GENAI_API_KEY=your_api_key
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
